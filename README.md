# README

Playing with the unofficial Qwant Music API

## INSTALL

- For simple users:
    ```
    git clone https://github.com/SebDeclercq/qwant_music_django.git .
    pipenv install
    ```
- For developers:
    ```
    git clone https://github.com/SebDeclercq/qwant_music_django.git .
    pipenv install --dev
    ```
